# Hatopia Website

Einfache, barrierefreie Beispiel-Webseite für den Verein Hatopia.

## Features
- Mehrsprachige Inhalte (Deutsch, Englisch, Türkisch, Russisch, Ukrainisch)
- Strukturierte Bereiche (Aktuelles, Angebote, Projekte usw.)
- Veranstaltungskalender mit abonnierbarer **ICS**-Datei
- Responsives, modernes Layout

## Entwicklung
Die Seite besteht aus statischem HTML, CSS und JavaScript und kann mit jedem
beliebigen Webserver ausgeliefert werden.

Zum Testen lokal ausliefern:

```bash
npx serve .
```

Die ICS-Datei befindet sich unter `events/events.ics` und kann in Kalender-Apps
abonniert werden.
